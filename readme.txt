Projekt je rađen u Code::Blocks-u tako da je potrebno samo otvoriti i buildati.
Build neće proći ako nije uključen c++11 standard, koji se može uključiti na:
Settings > Compiler > Compiler Settings > Compiler flags checkiranjem opcije c++11

Prilozen je i izvrsni fajl koji se moze direktno pokrenuti duplim klikom.
